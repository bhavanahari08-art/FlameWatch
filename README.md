# FlameWatch
FlameWatch – Intelligent LPG Safety Monitoring System

FlameWatch is an IoT-based safety system that detects LPG leaks, monitors fuel usage, and automatically prevents hazardous situations in real time.

Overview
The system integrates gas concentration sensing, cylinder weight monitoring, automated regulator control, cloud connectivity, and AI-driven analytics to enhance safety, efficiency, and preventive maintenance.

Key Features
Continuous LPG leak detection
Automatic regulator shutoff on unsafe gas levels
Cylinder weight monitoring for usage tracking and anomaly detection
Real-time cloud data transmission via embedded microcontroller
AI-based predictive alerts and false-positive reduction
Multi-channel alerts (audible, mobile, emergency escalation)
System Operation

Sensors continuously monitor ambient gas concentration and cylinder weight. When gas levels exceed a predefined safety threshold, the system autonomously shuts off the regulator to stop gas flow immediately. All data is transmitted to a cloud platform for monitoring and analysis.

Platform
A mobile and web interface provides:
Real-time monitoring
Historical usage analytics
Safety status visualization
Alert notifications
Intelligent Capabilities
Predictive refill estimation
Leak pattern detection
Sensor anomaly identification
Remote diagnostics

Applications:
Residential kitchens
Commercial and industrial kitchens
LPG distribution systems
Value Proposition

FlameWatch provides a scalable and cost-effective solution that combines real-time sensing, automated control, and AI-driven insights to reduce the risk of gas leaks, explosions, and fuel wastage.
